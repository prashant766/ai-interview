# AI-Interview Backend

A Java-based backend for AI-driven interview prep. Powers question generation and scoring.

## Quick Start
1. Install Java 17+ and Maven.
2. Run `mvn clean install`.
3. Start with `mvn spring-boot:run` (once you add Spring Boot deps).
4. Docker: `docker build -t ai-interview . && docker run -p 8080:8080 ai-interview`

## Tech
- Java 17
- Maven
- Spring Boot (planned)
- Integrates with frontend at [your-frontend-repo].
